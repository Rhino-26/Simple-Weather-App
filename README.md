🌤️ Weather App

A basic and user-friendly Weather Application built using HTML, CSS, and JavaScript that fetches real-time weather data from a public API such as OpenWeatherMap. This app allows users to search for any city and instantly view current weather conditions in a clean and modern interface.

The application dynamically retrieves weather data using JavaScript and displays key details like temperature, weather condition, humidity, wind speed, and an appropriate weather icon. It also includes error handling for invalid city names and a loading indicator to enhance user experience during data fetching.

Designed with simplicity in mind, the app features a responsive layout that works smoothly across mobile, tablet, and desktop devices, making it accessible and easy to use for beginners.

Core Features:

- Search weather by city name
- Real-time data fetching from a public API

Display of:
- City name
- Current temperature
- Weather condition (e.g., sunny, cloudy, rain)
- Humidity
- Wind speed
- Dynamic weather icon
- Error handling for invalid inputs
- Loading indicator while fetching data

API Integration:

- Uses a free weather API (e.g., OpenWeatherMap)
- Fetches data via JavaScript (Fetch API or Axios)
- Parses and displays JSON data dynamically

UI/UX Highlights:

- Clean and modern interface
- Fully responsive design (mobile, tablet, desktop)
- Smooth user experience with loading feedback

Optional Enhancements:

- Automatic location detection using Geolocation API
- Toggle between Celsius and Fahrenheit
- Dynamic background changes based on weather conditions
- Save last searched city using local storage

This project is beginner-friendly, well-structured, and easy to understand, with properly commented code to help learners grasp the fundamentals of API integration and dynamic UI updates.
